## Requirements

- Docker
- Docker Compose

## Commands

### Docker Compose Startup

```bash
docker-compose up --detach
```

### Dependencies Installation

```bash
docker-compose exec node npm install
```

### Server Startup

```bash
docker-compose exec node npm start
```

### Docker Compose Shutdown

```bash
docker-compose down
```
